# 📚 Library Management System

## 📌 Project Overview
This is a simple Java project that demonstrates the use of **Java Collections Framework** and **Exception Handling**.  
It simulates a basic library system where books and categories are managed using different collection types.

---

## 🎯 Objectives
- Learn and implement Java Collections
- Understand Exception Handling in Java
- Create and use Custom Exceptions
- Practice real-world mini project structure

---

## 🛠️ Technologies Used
- Java
- VS Code
- Git & GitHub
